# Collaborative Drawing Canvas

A real-time collaborative drawing application where multiple users can draw simultaneously on the same canvas with live synchronization.

## Features

- **Real-time Drawing**: Draw with brush and eraser tools with instant synchronization
- **Multi-user Support**: See other users' cursors and drawings in real-time
- **Color Palette**: Choose from 8 different colors
- **Brush Size Control**: Adjustable brush and eraser sizes
- **Undo/Redo**: Local undo/redo functionality
- **Save/Load**: Save your drawing as PNG or load images
- **User Management**: See who's online with colored indicators
- **Mobile Support**: Touch-enabled for mobile devices
